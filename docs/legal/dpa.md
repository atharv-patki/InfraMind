# Data Processing Addendum (DPA) - Draft

Last updated: 2026-03-10

## Roles
- Customer: Data Controller
- InfraMind AI: Data Processor

## Processing Scope
- Processing limited to providing contracted monitoring and incident-management services.

## Security Measures
- Access controls
- Encryption in transit
- Audit logging of sensitive operations

## Sub-processors
- Cloud provider and notification providers as disclosed in sub-processor register.

## Data Subject Requests
- Processor assists controller in responding to access, deletion, and correction requests.

## Breach Notification
- Processor notifies controller without undue delay upon confirmed personal data breach.

## Cross-Border Transfers
- Transfer mechanisms applied as required by jurisdiction.

## Legal Review Status
- Draft prepared and ready for legal approval and contract finalization.
